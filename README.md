# 📱 Smart Attendance System

A modern mobile attendance management application developed in Java to simplify and automate attendance tracking for educational institutions. The application provides an efficient way for lecturers to manage attendance records while enabling students to monitor their attendance history.

## 🚀 Features

* 👤 User Authentication (Login & Registration)
* 📚 Course/Class Management
* ✅ Student Attendance Check-in
* 📊 Attendance History
* 🔍 Search Students
* 📅 View Attendance by Date
* 👨‍🏫 Lecturer Dashboard
* 👨‍🎓 Student Dashboard
* 💾 Local Database Integration
* 📱 Clean and User-Friendly Interface

## 🛠️ Built With
